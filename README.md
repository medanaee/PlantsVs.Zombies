
# Plants Vs Zombies

This is an implementation of the famous game Plants vs. Zombies in C++ using the SFML library.

## Prerequisites

Before building and running the project, you must install the SFML library on your system.

### Installing SFML on Linux (Ubuntu/Debian)
Open your terminal and run the following commands:
```bash
sudo apt update
sudo apt install libsfml-dev
```

## How to build and run

Follow these steps to clone, compile, and run the game:

1. **Clone the repository:**
```bash
git clone https://github.com/medanaee/PlantsVs.Zombies.git
```

2. **Navigate into the project directory:**
```bash
cd ./PlantsVs.Zombies
```

3. **Build and run the game:**
```bash
make
```
